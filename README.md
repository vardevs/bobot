# bobot

> A GitHub App built with [Probot](https://github.com/probot/probot) that Bo does what needs to be done.

## Setup

```sh
# Install dependencies
npm install

# Run the bot
npm start
```

## Docker

```sh
# 1. Build container
docker build -t bobot .

# 2. Start container
docker run -e APP_ID=<app-id> -e PRIVATE_KEY=<pem-value> bobot
```
